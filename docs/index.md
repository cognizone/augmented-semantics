## Why Augmented Semantics?

Semantic Web technologies like SKOS, RDF, OWL, and SHACL are powerful but often require complex tooling. Augmented Semantics brings modern, user-friendly interfaces that run entirely in your browser.

### Key Features

- **No Backend Required** — Connect directly to SPARQL endpoints via HTTP
- **Privacy First** — Your data never leaves your browser
- **Modern UI** — Built with Vue 3, TypeScript, and PrimeVue
- **Dark Mode** — Easy on the eyes, day or night
- **Keyboard Shortcuts** — Power user productivity
