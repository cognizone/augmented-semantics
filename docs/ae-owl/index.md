# AE OWL

::: warning Planned
AE OWL is planned for future development. Documentation will be available once development begins.
:::

## Overview

AE OWL will be a browser-based OWL ontology viewer and navigator. It will help users understand and explore complex ontologies through intuitive visualizations.

## Planned Features

- **Class Hierarchy** — Navigate rdfs:subClassOf relationships
- **Property Browser** — Explore object and datatype properties
- **Axiom Viewer** — View class restrictions and equivalences
- **Visualization** — Interactive graphs of ontology structure
- **Import Analysis** — Track ontology imports and dependencies
- **Documentation View** — Render rdfs:comment and annotations

## Use Cases

- Exploring existing ontologies before reuse
- Understanding domain models in knowledge graphs
- Documenting ontology design decisions
- Teaching OWL concepts to newcomers
